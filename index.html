Creation Guide

제작 가이드

고정된 대본을 드리지 않습니다. 설명하기 좋고 촬영하기 좋은 몇 가지 방향을 제시해 드리니, 본인 채널의 성격에 맞게 자유롭게 수정하세요.

Direction

방향 제안 (1~2가지를 자유롭게 선택하여 사용하세요)

🇯🇵 방향 A
「일본어를 몰라도 연재 원서를 볼 수 있어요」
추천 대상: 일본 만화 / Pixiv / 소년 만화 관련 채널 · 숏폼 영상 60~120초
콘텐츠 구성 제안
①
본인이 즐겨 보는 일본 만화 원서를 찾습니다 (팔로워들에게도 익숙한 IP 추천)
②
번역을 켜지 않은 상태를 먼저 보여줍니다: 화면 가득한 일본어, 읽기 힘든 모습
③
Immersive Translate 만화 번역 기능을 켠 모습을 보여줍니다：같은 페이지가「원본 이미지 + 한국어 번역문」말풍선 안에 들어갑니다（PC 브라우저또는 App 내장 브라우저모두 가능）
④
몇 페이지를 넘기며 번역이 잘 된 / 재미있는 대사를 읽어주고, 의성어 변환 효과도 함께 보여줍니다

「최근에 일본 연재 만화를 보고 있는데，제가 일본어 노베이스라，예전에는 번역본이 올라올 때까지 기다려야만 했어요——그런데 지금은 Immersive Translate를 써서，공식 사이트에서 직접 봅니다，AI 번역이 말풍선 위에 겹쳐지고，조차「ドカン！」같은 의성어도 번역해줘서「쾅!」，정말 몰입감이 좋아요。게다가 무료 사용자도 매월 100장의 페이지를 번역할 수 있어서，가볍게 몇 작품 챙겨보기에 충분합니다。」

콘텐츠 구성 제안
①
도입부에서 직접 불편한 점 언급: "공식 번역판이 아직 안 나왔거나 너무 늦게 올라오는 작품 있으시죠 —— 어떻게 해결하시나요?"

번역을 켜지 않은 상태를 먼저 보여줍니다: 화면 가득한 일본어/영어, 전혀 읽을 수 없는 상태

Quality Criteria

우리가 생각하는 '잘 만들어진 콘텐츠'의 기준

01
단순한 말하기 광고가 아닌, 실제 읽는 화면 포함
최소 1~2개의 만화 스크롤 장면 + 번역 켜기/끄기 전후 비교 장면을 넣어, 팔로워가 직관적으로 느낄 수 있게 해주세요: 번역 끄기 = 전혀 이해 불가; 번역 켜기 = 스토리 파악 가능.
02
팔로워가 한눈에 이해할 수 있게: 이 도구는 '원서를 못 읽는 사람'을 위한 것입니다
제품의 이미지는 '일반적인 번역 앱 광고'가 아니라, '원서 읽기 / 원작 보기 / 공식 업데이트 챙겨보기를 도와주는 도구'여야 합니다.
03

📱 채널 링크 발송 👥 팔로워 규모 안내 📋 콘텐츠 방향 설명

자료 발송

🔗 협업 참고 페이지 🏷️ 팔로워 전용 할인 코드 🎁 Pro 무료 체험 계정

계약 체결 & 선금 지급
협업 계약서 서명 후 선금 지급
협업 방향과 조건이 확정되면 간단한 협업 계약서를 체결합니다. 계약서에는 콘텐츠 요건, 게시 일정, 보수 조건, 수정 횟수 등이 명시됩니다. 계약서 서명 완료 즉시 선금을 지급합니다.
📄 협업 계약서 체결 💰 보수 조건 확정 💸 선금 즉시 지급

🔗 댓글 상단 고정 링크 & 할인 코드 🎁 추첨 주간 이용권 준비 완료
